Como Crear un blog con Html, Css y JavaScript

HTML

Por David Mengibar - 5 de noviembre, 2025

HTML son las siglas de HyperText Markup Language (Lenguaje de Marcado de Hipertexto), y es el lenguaje de marcado estándar que se utiliza para estructurar y dar formato al contenido de una página web.

CSS

Por David Mengibar - 6 de noviembre, 2025

CSS son las siglas de Hojas de Estilo en Cascada (del inglés Cascading Style Sheets), un lenguaje utilizado para dar estilo y definir la apariencia de los documentos web, principalmente los escritos en HTML.

JavaScript

Por David Mengibar - 7 de noviembre, 2025

JavaScript es un lenguaje de programación interpretado que se utiliza principalmente para crear páginas web interactivas y dinámicas.
